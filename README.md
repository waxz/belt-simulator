# Conveyor flow simulator

## This is a pshysics sumulator of logistic conveyor system for PLC algorithm developemnt.

## Feature
- **Visual Editor**: User can design the the layout of belt, sensor and other equiments by configuration panel and drag interactive. The design configuration can be imported and exported.
- **Physics Simulation**: The physics simulation make the parcels move on belt and trigger sensor.
- **PLC communication**: Through the data exchange, PLC can get the status of the belt, sensor and equipment and send command to control the running status of belt and other equipments.



## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
